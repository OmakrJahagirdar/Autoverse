##  Overview

**AutoVerse** is a modern, responsive multi-page automotive manufacturing website designed to showcase a company's products, services, history, markets, careers, and contact information.

The website features a professional UI with smooth animations, dropdown navigation, responsive layouts, and an attractive automotive-themed design suitable for industrial or manufacturing businesses.

---

##  Features

* Modern and responsive design
* Multi-page website structure
* Animated hero sections
* Dropdown navigation menus
* Mobile-friendly navigation
* Automotive-themed UI
* Smooth hover effects and transitions
* Careers section with expandable content
* Company history and vision pages
* Product and market showcase
* Contact page with call-to-action

```bash

```

3. Open `index.html` in your preferred browser.

No additional installation or dependencies are required.
